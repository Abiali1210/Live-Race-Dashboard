## Field Mapping

Each `vertical.entrylist` item is mapped as follows:

| `CarMetadata` field | Snapshot field |
|---|---|
| `carNumber` | `entry.car_number` |
| `entrantId` | `entry.id` converted to string |
| `teamName` | `entry.team_name` |
| `teamColor` | `entry.team_color` |
| `className` | `entry.car_class` |
| `make` | `entry.car_make` |
| `model` | `null` |
| `drivers` | `entry.drivers[*].full_name` |
| `carshotUrl` | `entry.carshot_url` |
| `carshotUrlFull` | `entry.carshot_url_full` |
| `headshotUrls` | `entry.drivers[*].headshot_url`, if present |

Entries without a usable `car_number` are skipped instead of crashing the backend.
