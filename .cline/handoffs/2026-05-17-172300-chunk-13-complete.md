## Potential Gotchas / Risks

- `TrackMap` depends on the local SVG structure and `extractTrackGeometry()` path indexes. If the source SVG changes, verify the component.
- The local track SVG has licensing obligations. Keep attribution visible:
  `Track outline © Pitlane02, Wikimedia Commons, CC BY-SA 3.0 / GFDL.`
- `TimingCar.metadata` is optional; UI must continue to tolerate missing metadata and missing images.
- The dashboard table is not virtualized yet. For ~160 cars this is fine, but virtualization may be useful later if row count or refresh complexity increases.
- The frontend API types are manually mirrored from backend response shapes. If backend shapes change, update `frontend/src/api/types.ts`.
- Backend startup creates a real WIGE websocket connection. Avoid starting/stopping backend casually while the recorder is running unless intentionally testing.
- There are untracked local `.cline/skills/` files and `skills-lock.json`; do not commit them unless explicitly intended.
